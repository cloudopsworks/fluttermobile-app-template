# JetSnack Flutter

A Flutter application template targeting Android and iOS, inspired by the Android JetSnack sample.

## Getting Started

1.  **Prerequisites:**
    - Flutter SDK installed and configured.
    - Android Studio / Xcode for device emulation/deployment.

2.  **Run the app:**
    ```bash
    flutter run
    ```

3.  **Run tests:**
    ```bash
    flutter test
    ```

## Features
- Home screen with a list of snacks.
- Detailed view for each snack.
- Basic shopping cart functionality.
- Material 3 theme.
